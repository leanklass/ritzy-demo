Ritzy Editor Demo

Ritzy is an open source, cross-browser (IE10+ and evergreen browsers), rich text, real-time collaborative editor for the web! Ritzy is built with Google Docs style real-time collaborative editing support from the ground up.

You can create a new editor instance. You will be redirected to a new URL that is specific to the created editor. That URL can be opened in another window or shared with buddies.
